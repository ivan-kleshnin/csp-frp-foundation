# Sleep

Pause task for defined time.

## sleep

### Desired behavior

```js
go(async () => {
  console.log(1);
  await sleep(1000);
  console.log(2);
});
```

```
1 (pause 1 s)
2
```

### Analogy

No

### API

```js
Number -> Promise ()
```

### Implementation

```js
let sleep = curry((timeMs) => {
  return new Promise(resolve => setTimeout(resolve, timeMs));
});
```
